应用端作为实际功能的开发平台，通过[通信协议](../comm/)，调用实现端的[应用协议](../proto/)进行开发。

我们提供了[应用端插件](../plugin/)可快速使用 TypeScript 语言进行开发。

若使用其他语言，则需要自己实现[通信协议](../comm/)。
